# Vercel Skills Wrapper

Auto-updating wrapper for [vercel-labs/agent-skills](https://github.com/vercel-labs/agent-skills).

## Install
```bash
/plugin marketplace add ChampPABA/vercel-skills-wrapper
/plugin install vercel-skills@vercel-skills-wrapper
```

## Update
```bash
/plugin update vercel-skills
```

## Skills Included
- `react-best-practices` - React/Next.js performance (40+ rules from Vercel Engineering)
